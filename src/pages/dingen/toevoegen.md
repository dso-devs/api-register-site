## Dingen toevoegen aan het Ding-register

De Dingen-api is een voorbeeld API om de demo te vullen met data. De data staat in de volgende [GitHub repository](https://github.com/developer-overheid-nl/dingen-api). Er is geen mogelijkheid om via de API zelf data toe te voegen. Wel is het mogelijk om zelf data toe te voegen door een pull request in te dienen op de genoemde repository.

## Disclaimer

De data is volledig fictief en met behulp van verschillende generatieve AI-tools gegenereerd.
Eventuele onjuistheden zijn niet met opzet toegevoegd. Deze mogen uiteraard wel gemeld worden of verbeterd worden.
